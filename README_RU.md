## 🌟 Основные возможности
- **🌍 10-языковая поддержка:** Полная локализация на RU, EN, TR, AR, DE, ES, RO, BG, SR, SQ.
- **📜 Динамическая система чеков:** Выбирайте из:
  - Термолента 80 мм / 50 мм (Оптимизирована для экономии бумаги)
  - Стандартный чек весов (11x24 см)
  - Профессиональный горизонтальный формат A5 (Landscape)
- **🦾 Администрирование и управление:** Расширенное управление запасами (Stok), фильтрация исторических данных и безопасное редактирование записей.
- **📦 Без установки (Portable):** Один файл `FreeKantar.exe` — просто скачайте и запустите.
- **🛡️ Безопасность данных:** Мгновенное резервное копирование и восстановление базы данных SQLite одним кликом.
- **🏷️ Профессиональный дизайн:** Чистые чеки без сторонней рекламы.

## 🔧 Готовность к корпоративным решениям
Разработанное на базе `.NET 8`, ПО FreeKantar оптимизировано для скорости и стабильности. Оно готово к интеграции:
- Подключение к **ERP/SAP/IFS**.
- Зеркалирование базы данных **SQL Server**.
- Распознавание номерных знаков на базе **ИИ** (LPR).

## 📥 Как начать?
1. Перейдите в раздел **[Releases](https://github.com/yourusername/FreeKantar/releases)**.
2. Скачайте последнюю версию `FreeKantar.exe`.
3. Подключение: Подключите весовой терминал через Serial (COM), установите скорость передачи данных (Baud Rate) и начните работу!
