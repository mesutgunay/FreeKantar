**FreeKantar** es un software para básculas de camiones de alto rendimiento, ligero y versátil, diseñado para el pesaje industrial moderno. Con soporte para **10 idiomas globales** y **4 formatos de recibo**, ofrece una experiencia fluida para operadores de todo el mundo.

## 🌟 Características Principales
- **🌍 Soporte Nativo en 10 Idiomas:** Totalmente localizado en EN, TR, AR, RU, DE, ES, RO, BG, SR, SQ.
- **📜 Sistema de Recibos Dinámico:** Elija entre:
  - Térmico de 80 mm / 50 mm (Optimizado para el ahorro de papel)
  - Recibo de Báscula Estándar (11x24 cm)
  - Formato Horizontal A5 Profesional
- **🦾 Gestión de Administrador:** Gestión avanzada de stock (Stok), filtrado de datos históricos y registros seguros.
- **📦 Sin Instalación (Portable):** Un solo archivo `FreeKantar.exe`, solo descargar y ejecutar.
- **🛡️ Seguridad de Datos:** Copia de seguridad y restauración instantánea de la base de datos SQLite con un solo clic.
- **🏷️ Recibos de Marca Blanca:** Diseño limpio sin publicidad de terceros.
