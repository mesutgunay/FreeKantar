## 🌟 Key Features
- **🌍 10-Language Native Support:** Fully localized in EN, TR, AR, RU, DE, ES, RO, BG, SR, SQ.
- **📜 Dynamic Receipt System:** Choose between:
  - 80mm / 50mm Thermal (Optimized for paper saving)
  - Standard Kantar Receipt (11x24 cm)
  - A5 Professional Horizontal (Landscape)
- **🦾 Admin Management:** Advanced stock (Stok) management, historical data filtering, and secure records.
- **📦 Zero Installation (Portable):** Single-file `FreeKantar.exe` - just download and run.
- **🛡️ Data Security:** Instant one-click SQLite database backup and restoration.
- **🏷️ White-Label Receipts:** Clean designing with no third-party branding.
